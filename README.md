# <にじいろゲーミング>

## サイト概要
### サイトテーマ
世の中のゲーミングデバイスに口コミ、評価をできるレビューサイト

### テーマを選んだ理由
ゲーミングデバイスが好きで、調べることが多いのですが、個人のレビューサイトのの評価が運営者の好みで評価が変わるため、皆が投稿でき、平等な評価を一覧で見ることができたらいいと思ったからです。

### ターゲットユーザ
ゲーム好き、ゲーミングデバイスを集めている人向け。

### 主な利用シーン
ゲーミングデバイスを購入しようと思った際に参考にする時。
